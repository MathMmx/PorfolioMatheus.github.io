I developed an image converter as part of the activities for the Machine Learning bootcamp offered by DIO. The project aims to convert images to black-and-white or grayscale formats, using appropriate techniques for image manipulation and processing. This application can be used in various scenarios, such as preprocessing data for machine learning models or meeting specific visualization needs. Developing this project strengthened my knowledge of image manipulation and the use of relevant libraries for the machine learning context.

Transfer Learning for Cat and Dog Classification

Another project challenge from the DIO bootcamp: Transfer Learning for cat and dog classification. The project uses two main models: a custom model with a convolutional network created from scratch and a model based on the VGG16 architecture with transfer learning and pre-trained weights. Tools used include Python, TensorFlow, Keras, and Matplotlib. The goal is to compare the performance of direct learning and transfer learning.
