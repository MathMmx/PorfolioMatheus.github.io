Conversor de Imagens

Desenvolvi um conversor de imagens como parte das atividades do bootcamp de Machine Learning oferecido pela DIO. O projeto tem como objetivo realizar a conversão de imagens para formatos em preto e branco ou em tons de cinza, utilizando técnicas adequadas para manipulação e processamento de imagens. Essa aplicação pode ser utilizada em diversas situações, como pré-processamento de dados para modelos de aprendizado de máquina ou para atender necessidades específicas de visualização. O desenvolvimento deste projeto reforçou meus conhecimentos em manipulação de imagens e no uso de bibliotecas relevantes para o contexto de aprendizado de máquina. .

Transfer Learning para Classificação de Gatos e Cachorros

Mais um dos desafios de projeto do bootcamp da DIO Transfer Learning para classificação de gatos e cachorros. O projeto utiliza dois modelos principais: um modelo customizado com rede convolucional criada do zero e um modelo baseado na arquitetura VGG16 com transferência de aprendizado e pesos pré-treinados. As ferramentas utilizadas incluem Python, TensorFlow, Keras e Matplotlib. O objetivo é comparar o desempenho entre aprendizado direto e transferência de aprendizado..
